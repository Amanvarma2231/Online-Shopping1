# Online Shopping System - Installation Guide

## Prerequisites
1. XAMPP or WAMP Server
2. Web Browser (Chrome, Firefox, etc.)
3. Text Editor (VS Code, Sublime, etc.)

## Installation Steps

1. **Install XAMPP/WAMP**
   - Download and install XAMPP from https://www.apachefriends.org/download.html
   - Or install WAMP from https://www.wampserver.com/

2. **Start Server**
   - Open XAMPP Control Panel
   - Start Apache and MySQL services

3. **Download Project**
   ```bash
   cd C:\\xampp\htdocs\
   git clone https://github.com/Amanvarma2231/Online-Shopping1.git
